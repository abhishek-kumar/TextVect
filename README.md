Prerequisites
================================

1. Make sure java is installed
 
        $ java -version
2. Make sure git is installed

        $ git --version

Setup
================================

1. Download this git repo and cd into the 'TextVect' directory

        $ git clone https://github.com/abhishek-kumar/TextVect
        $ cd TextVect
2. Run the install script

        $ chmod u+x ./script/*
        $ ./script/install.sh

Run
================================
Run TextVect

    ./script/run.sh
